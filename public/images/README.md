## Image Sourcing Recommendations

### Free Stock Photo Sites
1. **Unsplash** (unsplash.com)
   - Search: "mountain landscape", "forest", "farmer", "agriculture", "soil"

2. **Pexels** (pexels.com)
   - Search: "cannabis", "marijuana plant", "hemp", "farming", "mountains"

3. **Pixabay** (pixabay.com)
   - Search: "hemp field", "organic farming", "mountain sunset"

### Search Keywords
- Trinity County landscapes: "northern california mountains", "redwood forest", "emerald triangle"
- Farming: "organic farmer", "sustainable agriculture", "hand trimming", "craft cannabis"
- Nature: "mountain sunset", "forest landscape", "rich soil"

## Image Specifications

- **Hero images**: Minimum 1920x1080px (landscape)
- **Content images**: Minimum 1200x800px
- **Portrait (Elias)**: Minimum 800x1000px (portrait orientation)
- Format: JPG or PNG
- Optimize images before adding (compress to reduce file size)
