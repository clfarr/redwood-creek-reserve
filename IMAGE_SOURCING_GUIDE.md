## üì• How to Download & Add Images

### Step 1: Download from Unsplash
1. Click on the image you like
2. Click the **"Download free"** button (no account needed, but creating one gives you collections)
3. The image will download as a `.jpg` file

### Step 2: Download from Pexels
1. Click on the image you like
2. Click the **"Free Download"** dropdown
3. Select **"Original"** size for best quality
4. The image will download

### Step 3: Rename & Optimize (Optional but Recommended)
```bash
# If you want to compress images to reduce file size:
# Use online tools like tinypng.com or squoosh.app
# Aim for under 500KB per image
```

### Step 4: Add to Project
1. Save downloaded images to `/home/user/redwood-creek-reserve/public/images/`
2. Rename to match the required filename (e.g., `hero-mountains.jpg`)
3. Replace the existing `.jpg` placeholder files

### Step 5: Test Locally
```bash
cd /home/user/redwood-creek-reserve
npm run dev
# Visit http://localhost:3000 and check each page
```

### Step 6: Commit & Push
```bash
git add public/images/
git commit -m "Add high-quality stock photos for production site"
git push origin main
```
